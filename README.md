<h1>This is a game made for hackclub</h1>
<p>This was made for an event called horizons</p>
<p>There was no AI useage in this project</p>
<h3>Movement</h3>
<p>The game is played by clicking left to pick up then left click to drop</p>
<p>You will be dragging the blocks across the frame and they will not be able to go over other blocks</p>
<p>The blocks should move into the space if you are close enough so you do not need to align perfectly</p>
